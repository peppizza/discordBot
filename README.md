Two discord bots for the french fortress discord server

## Dependecies

Requires discord.py rewrite, and python-dotenv

`python3 -m pip install -U discord.py`

`pip install -U python-dotenv`

## Compilation

fellow_kids bot:

`python3 fellow_kids.py`

navi bot:

`python3 navi.py`
